#+TITLE: Magit Basics

Magit: =SPC g=
* git status: =SPC g g=
** move cursor to a commit, then:
dwim means 'do what I mean'
- diff popup: =d=
  + diff dwim: =d d=
- log popup: =l=
  + log dwim: =l l=
** move cursor to a hunk, then:
- stage: =s=
- unstage: =u=
- stage everything: =S=
- unstage everything: =U=
** select a region of changes or hunk: =C-SPC=
=s= and =u= also work
** commit popup: =c=
create a regular new commit: =c c=
** fetch popup: =f=
** pull popup: =F=
** push popup: =p= or =P=
